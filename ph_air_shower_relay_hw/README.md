# ph_pcr_cabinet_relay_hw
This is relay hardware for project PCR Cabinets.

# Description

## Features:
    * Điện áp ngõ vào 12VDC.
    * Hỗ trợ 3 ngõ ra kích hoạt thông qua các relay.
    * Hỗ trợ 5 cổng output mở rộng kết nối với các thiết bị ngoại vi.
